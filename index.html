About Me

I am currently a third year student within the Game Design and Development major at Rochester Insitute of Technology. I have experience with programming with C# and C++. And I also have experience with HTML, CSS and JavaScript for web based development. Though my work I have also gained profieciency in Unity, as well as Autodesk Maya.

Winter's Dawn

Winter's Dawn is solo academic project made over a few weeks, made using PIXI-JS, HTML-5 and CSS. The premise of this game is that the player must survive as long as they can, racking up score by advancing. I added in snow, ambience related to wind, and sfx that matched up closely to the actions of the player. There is also a fairly robust state machine associated with the player, which allows the player to heal, attack, block, move or roll.

Desert Temple

In another class project, I used Autodesk Maya to create 3D-assets, then I UV-unwrapped them and applied textures using Adobe Substance Painter. I imported the assets into Unity, combining them with a terrain system, fps controller, skybox, and sound systems to create a walkable scene.

Test of Wills

Test of Wills was a class group project that I worked on in a team of 4. Within this game the player must survive against different bosses in a bullet-hell environment. The game was programmed in C#, using Monogame as the engine. I organized and led the group, and designed the architecture of the game. Throughout the project, I specialized in programming gameplay, including game states, and data loading and saving.
