Winter's Dawn

Winter's Dawn is solo academic project made over a few weeks, made using PIXI-JS, HTML-5 and CSS. The premise of this game is that the player must survive as long as they can, racking up score by advancing and killing enemies. All of the assets within the game were externally gathered.

Autumn Idle

The game was made over the span of 20 hours in a team of 5 for a Halloween game jam. It is a idle pumkin farming game made with Unity. I was responsible for everything regarding the critters within the game(crows and squirrels) aside from the art. The critters try to eat the pumpkins as they are growing. I also implemented the sounds and music into the game using Unity Scriptable Objects and events. Note: The WEBGL build is a bit finicky, as this was submitted for a game jam: some of the build bugs will be fixed at a later date.

Test of Wills

Test of Wills was a class group project that I worked on in a team of 4. Within this game the player must survive against different bosses in a bullet-hell environment. The game was programmed in C#, using Monogame as the engine. I organized and led the group, and designed the architecture of the game.
