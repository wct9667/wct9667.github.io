Cosmic Wanderer

About Cosmic Wanderer

Tools Used: Unity, C#, Android OS.

Project Length: 1 semester (September 2024 - December 2024).

Overview: Cosmic Wanderer is an AR stargazing and a constellation drawing application. Using a realistic starmap, the application accurately maps the stars to the users location. It also has functionality to change location, so the user can view stars from anywhere they wish. App APK can be shared upon request and is still being developed.
