Scan Effect

About the Scan Effect

Tools Used: Unreal, Blueprints, Materials, Post-Processing.

Project Length: 2 weeks.

Overview: This is a scan effect created in UE5 using blueprints, dynamic materials and post processing. The system is dynamic, attaching a post process element at runtime, blending the effect in and out based on a speed. Multiple post process functions are utilized, to both emulate the pulse effect, outline the world, and create a wallhack.

Below exists a devlog with more information about how it works. A zip of the files to reference, as well as the reference video for the pulse effect.
