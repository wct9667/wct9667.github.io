Echo Breach

About Echo Breach

Team Size: 5

Tools Used: Unity C#, Unity Shaderlab.

Project Length: 1 semester (September 2024 - December 2024).

Overview: Echo Breach is a 3d horror game in Unity. 2 players find themselves on a spaceship, destined to crash into the planet below. They must work together to escape and survive.

Contributions:

  • Programming gameplay features.
    • Flashlight.
    • Transparency System for walls.
    • Interaction System.
    • Push Puzzle.
    • Doors and locks.
  • Lighting.
  • UI Programming
    • Flashlight UI
    • Diegetic Screens (Timer and Planet)
    • Interaction Prompts
    • Main Menu: Planet Scaling, Light, Volume Slider
  • Post Processesing.
  • Shaders: Static for television screens and transparency.
